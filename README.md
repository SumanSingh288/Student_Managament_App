# 🎓 Student Record Management System

A Command Line Interface (CLI) based application developed using Python to manage student records efficiently. The system allows users to add, view, search, update, and delete student information stored in a JSON file.

---

## 📌 Project Overview

The Student Record Management System is designed to simplify the management of student data through a menu-driven interface. It provides an easy way to maintain records such as student ID, name, age, subject, and marks while implementing CRUD (Create, Read, Update, Delete) operations.

---

## ✨ Features

- Add New Student Records
- View All Students
- Search Student by ID
- Update Existing Student Details
- Delete Student Records
- Store Data Using JSON Files
- Menu Driven CLI Interface
- Error Handling and Validation

## 🚀 How to Use

After running the program, a menu will appear:

```text
1. View Students
2. Add Student
3. Search Student
4. Update Student
5. Delete Student
6. Exit
```

Select an option and follow the instructions displayed on the screen.
